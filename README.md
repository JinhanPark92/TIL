# Java Programming Language

## Ch.01 변수

### 1. 변수란?

* 변수란, 단 하나의 값을 저장할 수 있는 메모리 공간.

1.  변수의 선언과 초기화
   * 변수의 초기화란, 변수를 사용하기 전에 처음으로 값을 저장하는 것.

### 2. 변수의 타입
* 기본형

  - 논리형, 문자형, 정수형, 실수형 계산을 위한 실제 값을 저장한다. 모두 8개.
* 참조형
  * 객체의 주소를 저장한다. 8개의 기본형을 제외한 나머지 타입.
